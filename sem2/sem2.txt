Директория - это нерегулярный файл, внутренний элемент дерева каталогов. 
Процесс - программа, запущенная на исполнение
Работа любой ОС построена на процессах. У каждого процесса есть идентификатор (PID). 
PID ограничивается разрядностью системы (это логично)
Каждый процесс может быть порожден только другим процессом (кроме kernel)-> процесс характеризуется ещё и Parents PID
Первый процесс, запускающийся при включении ОС - это процесс ядра с 
PID = 0
PID = 1 - init. После его создания kernel завершается

getpid - get process identification
Если запускаем из терминала, то его родитель - терминал. Соответственно для разных терминалов PPID тоже разный.
***Создание процессов***
Процессы в UNIX могут быть порождены с помощью системного вызова fork. Он создает точную копию того процесса, который его порождает. 
Это единственный способ создания процесса
