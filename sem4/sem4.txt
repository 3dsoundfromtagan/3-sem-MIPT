Pipe между НЕродственными процессами не работает
Работа через файл FIFO, к кторому должен быть подключён хотя бы один процесс,
иначе пайп удалится
mknod - create a special or ordinary file
mode - S_IFIFO|0666
dev - 0
чтобы не было рандома с правами доступа, (void)umask(0)
Файл открывается open, систаксис такой же
Но при открытии открывается не сам файл, а поток связи с пайпом и в себя
записывает адрес этого пайпа. Другой процесс может его считывать.
Не забывать закрывать!
Недостатки pipe'a:
1. Без доп. данных непонятно, какие данные из какого процесса
2. Раздельность чтения и записи
3. Для работы пайпа должен быть активен хотя бы 1 из сообщающихся процессов
